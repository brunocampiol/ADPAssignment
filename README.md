# Math Calculator

## Build Application

On command prompt:

1. Go to '\src\ADP.Assignment.Service'
1. Run command 'dotnet build'

## Run Application

On command prompt:

1. Go to '\src\ADP.Assignment.Service'
1. Run command 'dotnet run'

## Test Application

On command prompt:

1. Go to '\test\ADP.Assignment.Unit.Tests'
1. Run command 'dotnet test'
